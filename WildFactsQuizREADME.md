# Wild Facts Quiz (Web)

Single-file HTML/JS port of the original Python/Tkinter quiz.

## Run locally
Open `index.html` in any modern browser.

## Host on GitHub Pages
1. Create a new GitHub repo (e.g. `wild-facts-quiz`)
2. Commit `index.html` to the repo root
3. GitHub → **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
4. Visit the Pages URL GitHub shows you.

No build tools, no dependencies.
